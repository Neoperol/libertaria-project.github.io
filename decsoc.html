Pillars of a Decentralized Society

The Decentralized Society is based on many federated, self-governing communities following shared principles and core values that can follow their own charter.

  • Decentralized Communication

    Before we can do anything with someone else, we need to communicate in a privacy-enabled manner.

  • Decentralized Law

    We should be able to rely on common standards that we can choose to form a contract in a decentralized way.

  • Decentralized Production

    Decentralized production is important to circumvent pseudo-protective tariffs and generate wealth, without third-party involvement.

  • Decentralized Finance

    We need to be able to trade freely without intermediaries including banks, if we choose so.
